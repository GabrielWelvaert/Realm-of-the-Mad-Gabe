**Controls:**
  - Keyboard:
        - *WASD* - Movement
        - *f* - Escape to nexus
        - *1-8* - Use inventory item
        - *esc* - Main menu
  
  - Mouse:
      - *Click on world* - Shoot
      - *Click and drag item* - Move item
      - *Shift click item* - Use item
      - *Mouse hover item* - View item description
      - *Mouse scroll* - Mini map zoom adjustment
